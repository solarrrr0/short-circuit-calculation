# 短路电流计算

这是一个 Python 桌面应用，用于按工程设计中的简化等值方法，计算光伏电站以 **T接接入** 或 **直接接入** 方式接入系统时的系统总短路电流。

最终电流统一以 **kA** 输出并保留三位小数；中间量显示六位小数，便于工程复核。

## 运行方式（Windows）

安装 Python 3.10 或更高版本后，双击 `run_app.bat`；也可在当前目录执行：

```bat
python app.py
```

程序只使用 Python 标准库，无需执行 `pip install`。在高分辨率 Windows 屏幕上，程序会启用 DPI 感知，避免系统位图缩放造成字体模糊。

## 设计情形

界面顶部可选择以下情形：

- **T接接入**：系统由“系统电源侧变电站”提供大电源支撑；T 接线路另一端为“T接对侧变电站”。程序计算系统电源侧、T接对侧及光伏升压站高低压侧的短路电流。
- **直接接入**：光伏站经一段线路直接接入系统电源侧变电站。T接对侧输入区、对应计算结果和 Excel 行会隐藏；只计算系统电源侧及光伏升压站高低压侧的短路电流。

“系统电源侧”与“T接对侧”是输入区中的工程角色说明。最终结果、中间量和 Excel 文件优先显示用户填写的实际站名及电压等级，例如“南竹变110 kV侧”。

## 输入项目

- 高、低压侧额定电压分别用于光伏电流和结果命名；相应基准电压用于基准电流与高压侧基准阻抗换算。
- 光伏额定容量可采用两种互斥输入方式：
  - 由“光伏额定有功 + 功率因数”计算；
  - 直接输入已知的额定容量（MVA）。
- `XΣ`、`XΣ0` 为系统提供的正序、零序等值阻抗，按 **p.u.** 输入。
- 架空线路单位电抗按 **Ω/km** 输入。T接方式使用“光伏站至 T 接点”和“T 接点至系统电源侧变电站”两段长度；直接接入仅输入“光伏站至系统电源侧变电站”一段长度。
- 变压器短路电压按百分数输入，例如 `10.5` 表示 \(10.5\%\)。

## 界面与可读性

- 输入区与结果区之间设有可拖动的竖向分隔条；拖动时只显示分隔条位置，松开后才刷新控件宽度和自动换行，因此不会因逐像素重排造成明显卡顿。
- 左侧输入区的初始宽度足以显示“直接输入”；长字段名称会按可用宽度自动换行。
- 右侧输出表格全部居中显示。光伏站结果分为高、低压两个区块，避免把高压侧“两相接地短路电流”和低压侧“两相短路电流”混在同一列中。
- 系统电源侧、T接对侧以及光伏升压站输入区标题会随用户输入的额定电压实时变化，例如“系统电源侧变电站220 kV侧”。

## 计算公式

### 1. 基准量与光伏短路贡献

设系统基准容量为 \(S_B\)，对应电压等级基准电压为 \(U_B\)，则：

\[
I_B=\frac{S_B}{\sqrt{3}U_B}.
\]

光伏额定容量按所选输入方式确定：

\[
S_{\mathrm{PV}}=\frac{P_{\mathrm{PV}}}{\cos\varphi},
\]

或直接采用用户输入的：

\[
S_{\mathrm{PV}}=S_{\mathrm{PV,input}}.
\]

光伏额定电流和短路贡献为：

\[
I_{\mathrm{PV,N}}=\frac{S_{\mathrm{PV}}}{\sqrt{3}U_N},
\qquad
I_{\mathrm{PV,sc}}=1.5I_{\mathrm{PV,N}}.
\]

### 2. 110 kV及以上侧故障电流

对输入 \(X_\Sigma\)、\(X_{\Sigma0}\) 的系统侧，定义：

\[
k_0=\frac{X_{\Sigma0}}{X_\Sigma}.
\]

三相、单相接地短路电流系数为：

\[
K_3=\frac{1}{X_\Sigma},
\qquad
K_{1g}=\frac{3}{2X_\Sigma+X_{\Sigma0}}.
\]

两相接地短路系数为：

\[
K_{2g}=
\frac{
\sqrt{3}\sqrt{\left(0.5+k_0\right)^2+\left(\frac{\sqrt{3}}{2}\right)^2}
}{1+2k_0}.
\]

程序在该公式内部使用 \(0.5+k_0\)，但不再将其作为独立中间量保存或显示。

\[
I_{k,\mathrm{sys}}^{(3)}=I_BK_3,
\qquad
I_{k,\mathrm{sys}}^{(1g)}=I_BK_{1g},
\qquad
I_{k,\mathrm{sys}}^{(2g)}=K_{2g}I_{k,\mathrm{sys}}^{(3)}.
\]

每种故障类型的总短路电流为：

\[
I_{k,\mathrm{total}}=I_{k,\mathrm{sys}}+I_{\mathrm{PV,sc}}.
\]

### 3. 光伏升压站高压侧等值阻抗

高压侧基准阻抗为：

\[
Z_B=\frac{U_B^2}{S_B}.
\]

T接接入时：

\[
X_{\mathrm{line}}=
\frac{\left(L_{\mathrm{PV-T}}+L_{\mathrm{T-source}}\right)x'}{Z_B}.
\]

直接接入时：

\[
X_{\mathrm{line}}=
\frac{L_{\mathrm{PV-source}}x'}{Z_B}.
\]

两种情形均采用：

\[
X_{\Sigma,\mathrm{PV-HV}}=X_{\Sigma,\mathrm{source}}+X_{\mathrm{line}},
\]

\[
X_{\Sigma0,\mathrm{PV-HV}}=X_{\Sigma0,\mathrm{source}}+2.5X_{\mathrm{line}}.
\]

其中 \(2.5\) 仅作用于架空线路正序电抗；系统电源侧给定的零序等值阻抗直接相加，不乘以 \(2.5\)。

### 4. 光伏升压站低压侧

变压器电抗统一到系统基准容量：

\[
X_T=\frac{u_k\%\times S_B}{100S_N}.
\]

当 \(S_B=100\ \mathrm{MVA}\) 时：

\[
X_T=\frac{u_k\%}{S_N}.
\]

低压侧正序短路阻抗及短路电流为：

\[
X_{\Sigma,\mathrm{PV-LV}}=X_{\Sigma,\mathrm{PV-HV}}+X_T,
\]

\[
I_{k,\mathrm{sys}}^{(3)}=\frac{I_{B,\mathrm{LV}}}{X_{\Sigma,\mathrm{PV-LV}}},
\qquad
I_{k,\mathrm{sys}}^{(2)}=0.866I_{k,\mathrm{sys}}^{(3)}.
\]

低压侧不计算单相接地短路电流；输出中的最后一列称为“两相短路电流”。

## Excel导出

点击“导出结果到 Excel”后选择保存位置即可生成 `.xlsx` 文件。文件中包含：

- 表 1：T接接入时为系统电源侧与 T接对侧的两行结果；直接接入时仅保留系统电源侧一行；
- 表 2：光伏升压站高压侧与低压侧分成独立区块，高压侧使用“单相接地短路电流”“两相接地短路电流”，低压侧使用“两相短路电流”。

Excel 使用默认字体和默认颜色；所有标题、表头及表内文字均居中，数值保留三位小数。
